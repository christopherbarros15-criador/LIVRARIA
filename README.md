Projeto de Livraria

Sobre o Projeto
Este sistema de gerenciamento de livraria foi desenvolvido como parte da disciplina de Programação III, cursada no 3º ano do curso de Licenciatura em Computação.
O projeto teve sua base estruturada durante as aulas práticas, sob orientação docente, servindo como uma base sólida para o aprendizado de frameworks web. Após a implementação inicial em sala de aula, o sistema foi disponibilizado para que os alunos pudessem realizar melhorias e novas implementações de forma independente, consolidando os conhecimentos adquiridos na disciplina.

Tecnologias utilizadas:

Python 3.11
Django 5.2
SQLite
Django Jazzmin (tema do Admin)
